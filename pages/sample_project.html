Weather Project Overview

The Weather Application is a multi-platform app developed using .NET MAUI. It retrieves weather data in real time from the OpenWeatherMap API and displays information such as temperature, humidity, wind speed, and forecast in a simple, user-friendly interface. The app was designed to provide clean visuals and smooth interaction across both mobile and desktop devices.
